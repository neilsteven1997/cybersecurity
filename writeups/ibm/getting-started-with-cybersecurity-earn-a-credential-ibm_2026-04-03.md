## Cybersecurity and Data

About this learning activity

In this course, you'll explore cybersecurity, including its key elements and the threats it addresses. You'll learn about data privacy, including the different types of sensitive data and the consequences that organizations face when data breaches occur. Then, you'll discover how organizations evaluate data security using the CIA triad, an industry- standard information security model. You'll also learn about data security controls used to combat cybersecurity threats.


What you'll learn

After completing this course, you should be able to:

    Determine ways in which the three key elements of cybersecurity apply to common tasks
    Categorize the various types of sensitive data that organizations must protect
    Apply the CIA triad to real-world situations
