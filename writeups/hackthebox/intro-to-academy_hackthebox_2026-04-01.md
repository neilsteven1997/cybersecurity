# Intro To Academy   

---

The HTB Academy provides a guided and interactive learning environment focused on cybersecurity skills. Its goal centers on 
making the process enjoyable while delivering practical knowledge through hands-on exercises that reinforce each concept and 
challenge understanding with new scenarios. Modules typically end with a skills assessment that applies the covered material to 
a realistic situation. The platform structures content into digestible units starting with paths as the broadest category, each 
path divided into multiple modules that address self-contained topics, and modules further split into individual sections. 
Certain paths connect directly to certifications earned after completing the associated exam. I found the breakdown into smaller 
sections particularly effective for steady progress without feeling overwhelmed.

Modules serve as the main building blocks, each functioning as a complete course on one specific topic and containing several 
sections. Previewing any module reveals its summary, expected outcomes, and any prerequisites. Modules fall into categories of 
offensive, defensive, purple, and general, with difficulty ratings of fundamental, easy, medium, or hard. They also use a tier 
system based on cube cost to unlock, where tier 0 modules are effectively free since the cost and reward balance out, while 
higher tiers require more cubes but return a portion upon full completion. The tier level does not indicate difficulty. Annual 
subscribers access included modules at zero cube cost while still receiving the completion reward. Completion requires solving 
all exercises and marking every section as done, followed by the skills assessment to confirm mastery before advancing.

Sections represent the smallest units, each focusing on a narrow part of the module topic and intended for completion in one 
session. They come in two forms: theory sections that establish foundational knowledge without practical tasks, and interactive
sections that include examples, demonstrations, and end with questions or exercises for hands-on application. Practice is 
encouraged by reworking the demonstrated steps. The Pwnbox provides a convenient in-browser Linux environment based on Parrot OS,
preloaded with relevant tools, accessible directly from interactive sections to follow along or solve tasks without local setup.

Exercises appear at the end of most interactive sections to test retention beyond the demos. They usually involve spawning a 
Docker or virtual machine target, with access details provided in the format of an IP and port or full credentials. Only one 
target runs at a time per user, with a visible timer that can be extended. Docker targets spawn quickly and require no virtual 
private network, allowing immediate browser access for web-based tasks. Virtual machine targets suit more complex needs like 
Windows or Active Directory environments and may involve a virtual private network connection if not using the Pwnbox. Questions 
include optional hints, and correct answers award cubes that accumulate toward unlocking further content. Optional exercises can
be revealed without blocking section completion, while required ones must be solved.

Paths group related modules around a shared theme, either as skill paths for focused techniques or job-role paths designed to 
prepare for specific cybersecurity positions. Enrolling in a path helps track progress on the dashboard without any cube cost or
automatic unlocking of modules. Job-role paths link to certifications that require passing a multi-day practical exam after 
finishing the path. The estimated duration listed for each path sums the time needed for its modules assuming eight-hour workdays.

Help resources include the top-bar help button for general platform questions, detailed step-by-step solutions for exercises 
available to annual subscribers, and the Hack The Box Discord server for community support. After finishing the introductory 
material, the next recommended step depends on experience level. Beginners should start with the Junior Cybersecurity Analyst 
path leading to the HTB Certified Junior Cybersecurity Associate certification. Those with some background or after earning 
that entry certificate can move to intermediate job-role paths. Experienced learners or those holding prior certifications 
advance to specialized options such as Senior Web Penetration Tester or Active Directory Penetration Tester paths.

### Key Takeaways
- Break learning into paths as the largest unit, then modules for specific topics, and sections as the smallest focused parts
- Preview any module on the modules page for summary, outcomes, and prerequisites before starting
- Categorize modules as offensive, defensive, purple, or general and rate them by difficulty as fundamental, easy, medium, or
  hard
- Unlock modules using cubes according to their tier, with 20 percent of spent cubes returned on 100 percent completion
- Complete a module by solving every exercise, marking all sections done, and passing the skills assessment
- Mark a section complete using the dedicated button after addressing any included questions
- Use theory sections to build foundational knowledge and interactive sections for practical exercises and demos
- Spawn the Pwnbox from any interactive section to practice examples and solve tasks in a preconfigured Parrot OS environment
- Spawn Docker or virtual machine targets for exercises, access them via provided IP and port or credentials, and extend timers
  as needed
- Solve exercise questions to earn cubes, using hints when available or revealing answers for optional ones
- Enroll in skill paths for targeted techniques or job-role paths for certification preparation without cube cost
- Follow the Junior Cybersecurity Analyst path as a beginner to earn the HTB Certified Junior Cybersecurity Associate
  certification
- Advance to intermediate or advanced job-role paths after gaining initial experience or certifications
- Access help via the top-bar button, subscription-based solutions, or the Hack The Box Discord server for questions or
  discussions
