# PH NIST CSF 2.0 Hypothetical Audits

**Project Overview**  
This repository contains hypothetical NIST Cybersecurity Framework 2.0 (CSF 2.0) audits for three Philippine community profiles:

- **Philippine Government**  
- **Private Companies**  
- **Filipino People** (citizens/community)

**Purpose**  
Demonstrate how NIST CSF 2.0 can be tailored to different Philippine contexts using real-world laws (Data Privacy Act, NPC guidelines, etc.).

**Repository Structure**
- `/profiles/` – One folder per profile (contains audit, recommendations, evidence, and this README)
- `main-research.md` – Full research background, methodology, and sources
- `/reports/` – Consolidated summary (optional)

**Framework Used**  
NIST Cybersecurity Framework 2.0 (2024) – Official Core + Profiles guidance (SP 1301)
