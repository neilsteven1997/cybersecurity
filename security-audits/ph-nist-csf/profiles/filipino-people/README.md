# Filipino People Profile

**NIST CSF 2.0 Community Profile**  
Hypothetical audit for everyday Filipino citizens and households (personal cybersecurity).

**Scope**  
Designed for non-technical users – focuses on personal devices, social media, online banking, and basic awareness (no enterprise controls).

**Files in this folder**
- `audit-filipino-people.md` – Full NIST CSF assessment (simplified for community level)
- `recommendations.md` – Practical action plan for individuals
- `evidence/` – Supporting notes and references

**Key Focus Areas**  
IDENTIFY (personal assets), PROTECT (passwords, 2FA), and basic awareness training.

Created as part of hypothetical security audit project – March 2026.
