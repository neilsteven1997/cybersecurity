# Botium Toys Audit

**NIST CSF 2.0 Security Audit**

This is my simple audit for Botium Toys – a small fictional toy company from the Google Cybersecurity course activity on Coursera.

**What it covers**  
A quick look at their basic security setup (online shop, customer data, daily operations) using the NIST Cybersecurity Framework 2.0.

**Files in this folder**
- botium-toys_2026-02-22.md – My filled-out NIST assessment
- recommendations.md – Easy fixes they should do
- evidence/ – My notes and screenshots

**Why I did it**  
Just to practice real-world auditing on a small business. Super helpful for understanding NIST in a simple way.

Done as part of my cybersecurity studies – February 22, 2026
